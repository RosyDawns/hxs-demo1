# TabBar 图标说明

## 需要的图标文件

请准备以下 10 个图标文件（尺寸建议 81x81 px）：

### 首页
- `home.png` - 首页未选中图标
- `home-active.png` - 首页选中图标

### AI管家
- `ai.png` - AI管家未选中图标
- `ai-active.png` - AI管家选中图标

### 能量树
- `tree.png` - 能量树未选中图标
- `tree-active.png` - 能量树选中图标

### 消息
- `message.png` - 消息未选中图标
- `message-active.png` - 消息选中图标

### 我的
- `me.png` - 我的未选中图标
- `me-active.png` - 我的选中图标

## 临时方案

在没有图标文件的情况下，可以：
1. 从 iconfont 下载对应图标
2. 使用设计工具快速制作简单图标
3. 或暂时使用纯色占位图标

## 颜色规范

- 未选中颜色：#999999
- 选中颜色：#FF6B6B
